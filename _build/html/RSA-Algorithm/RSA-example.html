The RSA Algorithm#

The Ron Rivest, Adi Shamir, and Leonard Adleman (RSA) algorithm is a public-key cryptosystem that is widely used for secure data transmission. It is based on the principle that it is easy to multiply large prime numbers together, however it is extremely difficult to factor the product of two large prime numbers. The RSA algorithm is based on the mathematical properties of prime numbers and the Eulerâ€™s totient function.

Letâ€™s break down the mathematics behind the RSA algorithm:

  1. Key Generation:

    • Choose two distinct prime numbers \(p\) and \(q\).

    • Compute \(n\) as the product of \(p\) and \(q\):

    \[ n = p \times q \]

    • Compute the totient function \(\phi(n)\):

    \[ \phi(n) = (p-1) \times (q-1) \]

    • Choose an integer \(e\) such that \(1 < e < \phi(n)\) and \(e\) is coprime with \(\phi(n)\). Typically, \(e = 65537\) is chosen.

    • Compute the modular multiplicative inverse of \(e\) modulo \(\phi(n)\) to get \(d\):

    \[ d \equiv e^{-1} \mod \phi(n) \]

    • The public key is \((e, n)\) and the private key is \((d, n)\).

  2. Encryption:

    • Convert the plaintext message \(M\) into an integer \(m\) such that \(0 \leq m < n\).

    • Compute the ciphertext \(c\) using the public key \((e, n)\):

    \[ c \equiv m^e \mod n \]

  3. Decryption:

    • Compute the plaintext integer \(m\) from the ciphertext \(c\) using the private key \((d, n)\):

    \[ m \equiv c^d \mod n \]

    • Convert the integer \(m\) back to the original plaintext message \(M\).

def euler_phi(n):
    result = n
    p = 2
    while p * p <= n:
        if n % p == 0:
            while n % p == 0:
                n //= p
            result -= result // p
        p += 1
    if n > 1:
        result -= result // n
    return result

Eulerâ€™s Totient Function

Eulerâ€™s totient function, \(\phi(n)\), is crucial for generating the RSA keys. It is calculated using the formula:

\[ \phi(n) = (p - 1) \times (q - 1) \]

where \(p\) and \(q\) are prime numbers.

Example:

Given:

\[\begin{split} p = 97 \\ q = 101 \end{split}\]

Calculate \(n\):

\[ n = p \times q = 97 \times 101 = 9797 \]

Calculate \(\phi(n)\):

\[ \phi(n) = (97 - 1) \times (101 - 1) = 96 \times 100 = 9600 \]

Encryption

To encrypt a message \(M\), convert it to an integer \(m\) such that \(0 \leq m < n\). Then, use the public key \((e, n)\) to compute the ciphertext \(c\):

\[ c \equiv m^e \mod n \]

Suppose the plaintext message \(M\) is represented as the integer \(m = 1234\).

Given the public key:

\[ (e, n) = (17, 9797) \]

Calculate the ciphertext \(c\):

\[ c \equiv 1234^{17} \mod 9797 = 4362 \]
